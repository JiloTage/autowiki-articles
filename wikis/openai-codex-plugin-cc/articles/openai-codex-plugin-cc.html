openai/codex-plugin-cc

最終更新: 2026-04-01
openai/codex-plugin-cc（Codex Plugin for Claude Code）は、OpenAIが公式リリースしたClaude Code向けプラグインであり、AnthropicのAIコーディングアシスタントClaude Code上でOpenAI Codexの機能を直接呼び出せるようにするブリッジツールである。

概要

openai/codex-plugin-ccは、OpenAIが開発・公式リリースしたClaude Code向けプラグインである。「cc」はClaude Codeの略称であり、プラグイン名はそのまま「Codex Plugin for Claude Code」を意味する。このプラグインにより、Claude Codeのユーザーは既存のワークフローを維持しながら、OpenAI Codexの高度なコード解析・タスク委譲機能を利用できる。

GitHub上で4,200スター・160フォークを獲得しており、Codexプラグインの中で最も人気の高い公式プラグインとなっている。ChatGPTの無料プランを含むサブスクリプション、またはOpenAI APIキーがあれば利用可能である。

図1: codex-plugin-ccのブリッジアーキテクチャ。Claude CodeとOpenAI Codexを接続する。

主要機能

このプラグインはいくつかのスラッシュコマンドを提供しており、Claude Codeのワークフロー内でCodexの機能を呼び出すことができる。

スラッシュコマンド一覧

  • /codex:review — 標準的なコードレビューを実行する
  • /codex:adversarial-review — 選択した実装や設計決定に疑問を投げかける対抗的レビュー
  • /codex:rescue — バグ調査・修正タスクをCodexに委譲する
  • /codex:status — バックグラウンドジョブの進行状況を確認する
  • /codex:result — 完了したジョブの結果を取得する
  • /codex:cancel — 実行中のタスクをキャンセルする
  • /codex:setup — インストールの設定と確認を行う

バックグラウンドタスク

codex:rescueコマンドにより、Codexへのタスク委譲がバックグラウンドで実行される。これにより、Claude Codeのユーザーはメインのコーディング作業を続けながら、Codexに別のバグ修正や調査タスクを並列実行させることができる。

インストールと設定

このプラグインはClaude Codeプラグインマーケットプレイス経由でインストールできる。以下の要件が必要である。

動作要件

  • Node.js 18.18以降
  • ChatGPTサブスクリプション（無料プランを含む）またはOpenAI APIキー
  • ローカルにCodex CLIがインストール済みであること

インストール手順

Claude Codeのターミナルで以下のコマンドを順番に実行する：

/plugin marketplace add openai/codex-plugin-cc
/plugin install codex@openai-codex
/reload-plugins
/codex:setup

技術アーキテクチャ

プラグインのアーキテクチャは、Codexアプリサーバーをラップする形で実装されている。ローカルにインストール済みのCodex認証情報（.codex/config.toml）と設定ファイルをそのまま利用するため、追加の認証設定は不要である。

プラグインのマニフェストは.codex-plugin/plugin.jsonに格納されており、Claude Codeの標準プラグイン仕様に従っている。また、Claude Codeのフックシステムと統合しており、コードレビューゲートなどの自動化されたワークフローに組み込むことができる。

図2: codex-plugin-ccの内部コンポーネント構成

ユースケース

openai/codex-plugin-ccは、以下のようなユースケースで特に効果を発揮する。

  • デュアルエージェント開発: Claude Codeで設計・実装を行いながら、Codexで並列にコードレビューを実行する
  • バグ調査の委譲: 複雑なバグ修正をCodexに委譲し、メインの開発フローを妨げない
  • 対抗的コードレビュー: /codex:adversarial-reviewにより、設計上の盲点を発見する
  • CI/CDパイプライン統合: Claude Codeのフックシステムと組み合わせ、コミット前の自動レビューゲートを設置する

競争的背景

このプラグインのリリースは、AI開発ツール市場における興味深い競争の動きを示している。OpenAIが競合他社であるAnthropicのツール（Claude Code）向けのプラグインを公式リリースするという戦略は、市場シェアを持つプラットフォームへの統合を優先する実用主義的なアプローチを反映している。

Claude Codeはプラグインエコシステムを急速に拡大しており、競合他社もそのエコシステムへの参入を余儀なくされている。これは、現代のAI開発ツール間の協調と競争が共存する複雑な市場環境を示している。
