Gitサブモジュールを使ったスキル共有アプローチ

最終更新: 2026-03-30
Gitのサブモジュール機能を活用して、Claude Code Skillsを専用リポジトリからプロジェクトに組み込む方法。追加ツール不要でGit標準機能のみを使う、バランスの良いアプローチ。

概要

Gitサブモジュールアプローチは、Skills専用のプライベートリポジトリを作成し、各プロジェクトにGitサブモジュールとして組み込む方法だ。Skills共有方法の概要で紹介した4つのアプローチの中で、セットアップの簡便さとバージョン管理能力のバランスが最も優れている。

Gitサブモジュールの基本

Gitサブモジュールは、一つのGitリポジトリを別のGitリポジトリのサブディレクトリとして埋め込む機能だ。親リポジトリはサブモジュールのコミットハッシュを記録するため、特定バージョンへの固定が可能となる。

図1: Gitサブモジュールによるスキル共有の構造図

セットアップ手順

Skills専用リポジトリの作成

まず、GitHubにプライベートリポジトリ（例: `myorg/claude-skills`）を作成し、以下の構造でSkillsを配置する：

claude-skills/
├── README.md
└── commands/
    ├── deploy.md
    ├── review-pr.md
    └── generate-tests.md

プロジェクトへのサブモジュール追加

# プロジェクトのルートで実行
git submodule add git@github.com:myorg/claude-skills.git .claude/shared-commands

# .gitmodules ファイルが自動生成される
git commit -m "Add claude-skills submodule"

シンボリックリンクの作成

mkdir -p .claude/commands

# 各Skillファイルをリンク
for skill in .claude/shared-commands/commands/*.md; do
  skill_name=$(basename "$skill")
  ln -sf "../shared-commands/commands/$skill_name" ".claude/commands/$skill_name"
done

git add .claude/commands/
git commit -m "Link shared skills to .claude/commands/"

更新ワークフロー

Skillsを最新版に更新

# サブモジュールを最新にする
git submodule update --remote .claude/shared-commands
git add .claude/shared-commands
git commit -m "Update claude-skills submodule to latest"

特定バージョンへのピン留め

# 特定タグ/コミットに固定
cd .claude/shared-commands
git checkout v1.2.0  # または特定のコミットハッシュ
cd ../..
git add .claude/shared-commands
git commit -m "Pin claude-skills to v1.2.0"

新規メンバーのクローン手順

# サブモジュールを含めてクローン
git clone --recurse-submodules git@github.com:myorg/myproject.git

# 既存クローンにサブモジュールを初期化
git submodule update --init --recursive

メリット・デメリット

Gitサブモジュールの最大のメリットはNode.jsやnpmが不要であり、Git標準機能だけで実現できる点だ。プロジェクトのリポジトリにどのバージョンのSkillsが組み込まれているかがGit履歴で追跡できる。一方で、`git clone` 時に `--recurse-submodules` フラグを忘れると初期化されないため、セットアップ自動化スクリプトでこの手順を自動化することが重要だ。また、サブモジュールの更新は自動では行われず、明示的な `git submodule update --remote` が必要となる。

高度な設定：.gitmodulesのカスタマイズ

[submodule ".claude/shared-commands"]
    path = .claude/shared-commands
    url = git@github.com:myorg/claude-skills.git
    branch = main  # 追跡するブランチを指定
