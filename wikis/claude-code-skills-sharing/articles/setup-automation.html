スキル配布のセットアップ自動化スクリプト

最終更新: 2026-03-30
Claude Code Skillsの配布・インストール・更新を自動化するスクリプトパターン集。開発者のオンボーディングを「ワンコマンド」に集約し、配布方法（npm/Gitサブモジュール/セントラルリポジトリ）に依らず適用できる。

概要

セットアップ自動化スクリプトは、選択した配布方法（npm / GitHub Packages、Gitサブモジュール、セントラルリポジトリのいずれか）に関わらず、開発者が「1コマンド」でClaude Code Skillsを利用可能にするための自動化層だ。

汎用インストーラースクリプト

どの配布方法にも対応できる汎用的なインストーラーの設計：

#!/usr/bin/env bash
# install-claude-skills.sh - 汎用Claude Code Skillsインストーラー
set -euo pipefail

SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
GLOBAL_COMMANDS_DIR="$HOME/.claude/commands"

usage() {
  cat <<EOF
Usage: $0 [OPTIONS]

Options:
  --mode=MODE       配布モード: submodule|central|npm (default: auto-detect)
  --global          グローバルインストール (~/.claude/commands/)
  --project=PATH    プロジェクトパスを指定
  --update          既存インストールを更新
  -h, --help        このヘルプを表示

Examples:
  $0 --global                        # グローバルインストール
  $0 --project=/path/to/my-project   # プロジェクトローカルインストール
  $0 --update --global               # グローバルインストールを更新
EOF
}

# 配布モードの自動検出
detect_mode() {
  if [ -f "package.json" ] && grep -q "claude-skills" package.json 2>/dev/null; then
    echo "npm"
  elif [ -f ".gitmodules" ] && grep -q "claude-skills" .gitmodules 2>/dev/null; then
    echo "submodule"
  else
    echo "central"
  fi
}

install_global() {
  local source_dir="$1"
  mkdir -p "$GLOBAL_COMMANDS_DIR"
  for skill in "$source_dir"/*.md; do
    [ -f "$skill" ] || continue
    local name=$(basename "$skill")
    ln -sf "$skill" "$GLOBAL_COMMANDS_DIR/$name"
    echo "  ✓ $name"
  done
}

# メイン処理
MODE=$(detect_mode)
echo "Mode: $MODE"
# ... (各モードに応じた処理)

Makefileによるタスク定義

プロジェクトにMakefileがある場合、Skillsのセットアップをmakeターゲットとして定義するとチーム全員が同じコマンドで操作できる：

# Makefile
.PHONY: setup-skills update-skills list-skills

## Claude Code Skills をセットアップ
setup-skills:
    @echo "Setting up Claude Code Skills..."
    @bash install-claude-skills.sh --global
    @echo "Done."

## Claude Code Skills を最新に更新
update-skills:
    @git submodule update --remote .claude/shared-commands
    @bash install-claude-skills.sh --update --global

## インストール済みのSkillsを一覧表示
list-skills:
    @ls -la ~/.claude/commands/*.md 2>/dev/null || echo "No skills installed"

dotfilesリポジトリとの統合

多くのエンジニアはdotfilesリポジトリを持っており、Claude Code Skillsのグローバルインストールをdotfilesのセットアップに組み込むことで、新マシンへの移行が容易になる：

# dotfiles/install.sh の一部
# Claude Code Skills のセットアップ
if command -v git >/dev/null 2>&1; then
  SKILLS_REPO="git@github.com:myorg/claude-skills.git"
  SKILLS_DIR="$HOME/.claude-skills"

  if [ ! -d "$SKILLS_DIR" ]; then
    git clone "$SKILLS_REPO" "$SKILLS_DIR"
  fi

  mkdir -p "$HOME/.claude/commands"
  for skill in "$SKILLS_DIR/commands/"*.md; do
    ln -sf "$skill" "$HOME/.claude/commands/$(basename "$skill")"
  done
  echo "Claude Code Skills installed."
fi

GitHub Actions による自動配布

Skills リポジトリに変更がpushされたとき、自動的に組織内の全リポジトリに通知する仕組み：

# .github/workflows/notify-update.yml（Skills リポジトリ側）
name: Notify Skills Update
on:
  push:
    branches: [main]
jobs:
  notify:
    runs-on: ubuntu-latest
    steps:
      - name: Post to Slack
        uses: slackapi/slack-github-action@v1
        with:
          payload: |
            {
              "text": "Claude Code Skills が更新されました。\n`make update-skills` を実行して最新版を取得してください。\nChangelog: ${{ github.event.head_commit.message }}"
            }
        env:
          SLACK_WEBHOOK_URL: ${{ secrets.SLACK_WEBHOOK_URL }}

Windows環境への対応

Windows環境ではシンボリックリンクに制限があるため、コピー方式またはPowerShellスクリプトを使う：

# install-claude-skills.ps1
$SkillsDir = "$env:USERPROFILE\.claude-skills"
$CommandsDir = "$env:USERPROFILE\.claude\commands"

New-Item -ItemType Directory -Force -Path $CommandsDir | Out-Null

Get-ChildItem "$SkillsDir\commands\*.md" | ForEach-Object {
  $dest = Join-Path $CommandsDir $_.Name
  Copy-Item $_.FullName $dest -Force
  Write-Host "Copied: $($_.Name)"
}

インストール検証スクリプト

#!/bin/bash
# verify-skills.sh - インストール状態を確認
echo "=== Claude Code Skills 確認 ==="
COMMANDS_DIR="$HOME/.claude/commands"

if [ -d "$COMMANDS_DIR" ]; then
  count=$(ls "$COMMANDS_DIR"/*.md 2>/dev/null | wc -l)
  echo "グローバルSkills: ${count}個"
  ls -1 "$COMMANDS_DIR"/*.md 2>/dev/null | while read f; do
    echo "  - /$(basename "${f%.md}")"
  done
else
  echo "Skills未インストール"
fi
