ブリッジシステム

最終更新: 2026-03-31
ブリッジシステム（src/bridge/、33ファイル）は Claude Code CLI と IDE 拡張機能（VS Code、JetBrains）間の双方向通信を実現する。JWT 認証によるメッセージセキュリティ・セッションライフサイクル管理・デバイス信頼管理などを実装し、IDE 上での Claude Code のシームレスな統合を可能にする。

ブリッジシステム概要

ブリッジシステムは src/bridge/ ディレクトリ（33ファイル）に実装されており、Claude Code CLI と IDE 拡張機能（VS Code、JetBrains）間のリアルタイム双方向通信を担当する。ユーザーが IDE 内から直接 Claude Code を操作できるようにする重要なインフラである。

ブリッジプロトコル

bridgeMain.ts（115.6 KB）がコアプロトコルを実装する。IDE 拡張機能からのリクエストを受信し、Claude Code CLI に転送する。レスポンスはリアルタイムでストリーミングされて IDE に返却される。

REPL ブリッジ

replBridge.ts（100.5 KB）は REPL（Read-Eval-Print Loop）モードのブリッジを実装する。インタラクティブな対話モードをサポートし、ユーザーが IDE のターミナルや専用パネルから Claude と直接対話できる。

JWT セキュリティ

jwtUtils.ts は JWT（JSON Web Token）を用いたメッセージの認証・検証を実装する。IDE 拡張機能と CLI 間の通信が改ざんされていないことを保証し、不正なクライアントからのリクエストを拒否する。

セッションライフサイクル

createSession.ts と sessionRunner.ts がセッションのライフサイクルを管理する。

  • 作成: IDE からのリクエストでセッションを開始し、認証情報を検証する
  • 実行: Claude との対話を管理し、ツール実行結果を IDE にストリーミングする
  • 終了: セッションのクリーンアップと状態の保存を行う

デバイス信頼管理

trustedDevice.ts はデバイスの信頼レベルを管理する。信頼済みデバイスからの接続は追加認証なしで処理され、未知のデバイスは認証フローを経る必要がある。

IDE 統合

ブリッジシステムは以下の IDE との統合を提供する。

  • VS Code: MCP の vscodeSdkMcp.ts との連携を含む
  • JetBrains: IntelliJ IDEA・PyCharm・WebStorm などの JetBrains 製 IDE をサポート

リモート機能

remoteBridgeCore.ts（39.4 KB）はリモートマシンでの Claude Code 実行をサポートするプロトコルを実装する。SSH 接続先やクラウドサーバーで Claude Code を実行しつつ、ローカルの IDE からシームレスに操作できる。
