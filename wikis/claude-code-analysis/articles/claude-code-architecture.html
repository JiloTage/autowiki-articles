主要なアーキテクチャの選択

TypeScript + React/Ink

TypeScript をメイン言語として採用し、ターミナル UI には React の Ink ライブラリを使用する。これにより宣言的な UI 構築とコンポーネントの再利用が可能になる。

Zod バリデーション

schemas/ ディレクトリに Zod スキーマを集約し、実行時の型バリデーションを徹底する。API レスポンス・設定ファイル・ユーザー入力などすべての外部データに適用される。

プラグインシステム

plugins/ ディレクトリにプラグインシステムを実装し、拡張性を確保する。サードパーティのツール・コマンド・エージェントをプラグインとして追加できる。

マルチエントリポイント

entrypoints/ に CLI・SDK・MCP・init の各エントリポイントを持ち、Claude Code を様々な方法で利用できる。
