MCP インテグレーション

最終更新: 2026-03-31
Claude Code は Model Context Protocol（MCP）を完全実装しており、src/services/mcp/ ディレクトリ配下に22ファイルで構成される。MCP クライアント（119 KB）・設定管理（51 KB）・OAuth 認証（88 KB）・接続管理・チャネルセキュリティ・公式レジストリ・VS Code SDK 統合が含まれる。外部ツールサーバーとの接続を標準化し、エコシステムの拡張性を実現する。

MCP 概要

Model Context Protocol（MCP）は AI システムが外部ツールやサービスと標準化された方法で通信するためのプロトコルである。Claude Code は src/services/mcp/ ディレクトリ（22ファイル）に完全な MCP 実装を持ち、外部ツールサーバーと接続してツールのエコシステムを拡張できる。

MCP クライアント

client.ts（119 KB）は MCP プロトコルの完全な実装を提供する。外部ツールサーバーへの接続・ツール一覧の取得・ツール呼び出しの実行・レスポンスの処理を担当する。MCP ツールは通常のビルトインツールと同じインターフェースで扱われ、ツールアセンブリ時にシームレスに統合される。

認証・OAuth

auth.ts（88 KB）は MCP サーバーへの認証フローを実装する。OAuth 2.0 に対応し、トークンの取得・更新・管理を行う。認証済みの MCP サーバーへのアクセスを安全に制御する。

サーバー設定管理

config.ts（51 KB）は MCP サーバーの設定を管理する。サーバーの追加・削除・更新・一覧表示の機能を提供し、設定はユーザー設定ファイルに永続化される。

チャネルセキュリティ

channelAllowlist.ts と channelPermissions.ts がチャネルレベルのセキュリティを実装する。特定のチャネルへのアクセスを許可リストで制御し、細粒度のパーミッション管理を実現する。

公式レジストリ

officialRegistry.ts は Anthropic が認定した公式 MCP サーバーのレジストリを管理する。ユーザーはレジストリから MCP サーバーを検索・インストールできる。

ツール統合

MCP ツールは src/tools.ts の assembleToolPool() でビルトインツールと統合される。ビルトインツールが MCP ツールより優先されるため、名前衝突時はビルトインが勝つ。MCP ツールはエージェントごとのツールセットに含まれ、エージェントの MCP 依存フィルタリングにより必要なサーバーが利用可能な場合のみロードされる。

接続ライフサイクル

MCP 接続は useManageMCPConnections.ts で管理される。接続は React フックとして実装されており、コンポーネントのライフサイクルと同期する。エージェント終了時には runAgent.ts のクリーンアップフェーズで接続が解放される。
